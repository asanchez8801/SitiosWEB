    • Proposito del Sitio

    El propósito de este sitio es aprender sobre las etiquetas FlexBox, listas y enlaces en CSS, se brindará inicialmente una prequeña descripción en que consiste cada uno de estos atributos. Despúes analizaremos su funcionamiento y posterioremente tendremos unas actividades para desarrollar y poner en práctica lo aprendido, todo esto lo podremos realizar desde este sitio web de una manera dínamica e intuitiva incentivando el uso de herramientas tecnologicas.

    FlexBox

    El FlexBox es una etiqueta que nos permitira alojar información dentro de una pagina web en casillas o columnas ya sean horizontales como verticales, basicamente lo que nos permite es maquetar todos los elementos de una pagina web separados independientemente como lo podemos observar en la siguiente imagén:
